#Basic React App
The whole purpose from this repo is to have the basic setup needed to start a react app and aslo to use webpack to transpile es6/7  code

Based on video: https://www.youtube.com/watch?v=HXOoe1VSKpo

Dependencies:
- React
- ReactDOM

Dev Dependencies:
- Babel-core
- Babel-loader
- Babel-preset-es2015
- Babel-stage-0
- Babel-preset-react
- Webpack
- Webpack-dev-server
